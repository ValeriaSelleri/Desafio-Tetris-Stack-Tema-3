# Desafio Tetris Stack – Tema 3
Trabalho de Estrutura de Dados — Estácio

Este repositório contém uma implementação didática do Tema 3: uso de **pilha (stack)** para gerenciar as peças colocadas em um tabuleiro tipo Tetris.

## O que implementa
- Pilha (LIFO) com push e pop.
- Tabuleiro 10x20 (simples, ASCII).
- Duas peças exemplo: `I` e `O`.
- Menu interativo em C para empilhar/desempilhar e visualizar o tabuleiro.
- Versão em Python para execução rápida.

## Como compilar (C)
Entre na pasta `src`:
```bash
cd src
make
./tetris_stack
